(as of jan 8, 2014, this is the proper procedure) - David L-B

1. sudo apt-get install python-pygame
2. sudo apt-get install pyqt4-dev-tools
3. go to http://www.pyqtgraph.org/ and install by clicking python-pyqtgraph_0.9.8-1_all.deb
4. cd to catkin workspace and run catkin_make
5. source devel/setup.bash
6. connect PS3 controller *(optional)

you are ready to launch the front-end
