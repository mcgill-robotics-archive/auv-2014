Performs more specific filtering and then finds the corners of any bins.

Parameters

A 2-Dimensional vector containing the corners of each marker. For each marker the corners are listed in clockwise order starting with the top-left or bottom-right corner when the bin is oriented vertically.

Detects the edges in the given image and uses them to draw a contour and approximate polygons for each contour.

Any contour with more than 4 edges and not convex is thrown away in order to find just contours matching the marker bins.
