ObjectData * Door::retrieveObjectData ( cv::Mat &  currentFrame) [virtual]

Applies the necessary filters to the current frame, checks if the door object is present and, if it is, extracts the necessary information.

Parameters:
currentFrameCurrent camera frame
Returns:
If the door is present, it returns a pointer to an ObjectData which contains the information gathered on the door. If the door is not present in the current frame, it returns the zero pointer (NULL).
