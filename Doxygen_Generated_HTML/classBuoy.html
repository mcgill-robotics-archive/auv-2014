std::vector< computer_vision::VisibleObjectData * > Buoy::retrieveObjectData ( cv::Mat &  currentFrame) [virtual]

Applies the necessary filters to the current frame, checks if the buoy object is present and, if it is, extracts the necessary information.

Parameters:
currentFrameCurrent camera frame
Returns:
If the buoy is present, it returns a pointer to an ObjectData which contains the information gathered on the buoy. If the buoy is not present in the current frame, it returns the zero pointer (NULL).
